# Federated Reinforcement Learning for UAV Swarm

## Description
This project implements UAV swarm coordination using PPO and Federated Learning in AirSim.

- V formation control
- Leader-follower mechanism
- Federated learning aggregation
- Fault tolerance

## Setup

## Run
